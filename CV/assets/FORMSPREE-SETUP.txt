============================================
  CONTACT FORM - Gmail par message kaise aaye
============================================

Formspree (free) use ho raha hai. Jab koi "Send Message" karega, 
aapko fatimachoudhry94@gmail.com par email aayegi.

STEP 1: Formspree account banao
--------------------------------
1. Browser mein open karo:  https://formspree.io
2. "Get Started" / "Sign Up" pe click karo
3. Sign up with Google use karke apna Gmail (fatimachoudhry94@gmail.com) se login karo

STEP 2: New form banao
--------------------------------
1. Dashboard pe "New Form" / "+ Create Form" pe click karo
2. Form name daal do (e.g. "Portfolio Contact")
3. "Email" field mein apna Gmail daalo jahan messages chahiye: fatimachoudhry94@gmail.com
4. Form create karo

STEP 3: Form ID copy karo
--------------------------------
1. Form banne ke baad aapko ek "Form Endpoint" dikhega, jaise:
   https://formspree.io/f/xyzabcde
2. Is URL ke end wala ID copy karo:  xyzabcde  (ye aapka alag hoga)

STEP 4: contact.html mein ID lagao (exact jagah)
--------------------------------
1. Portfolio folder mein  contact.html  open karo.
2. Ctrl+F se "YOUR_FORM_ID" search karo — ye form ke action attribute mein hai (line ~73).
3. Sirf "YOUR_FORM_ID" ko apna Formspree ID se replace karo (e.g. xyzabcde).
   Pehle:  action="https://formspree.io/f/YOUR_FORM_ID"
   Baad:   action="https://formspree.io/f/xyzabcde"
4. File save karo.

Ab jab koi contact page par message bhejega, aapko Gmail par email aa jayegi.
