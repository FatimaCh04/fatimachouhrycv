# Profile photo (live site ke liye)

**Vercel / live site par apni photo dikhane ke liye:**

1. Apni ek clear photo (square ya portrait) **profile.jpg** ke naam se is folder mein save karo.
2. Repo mein push karo (GitHub par upload karo).

Jab tak **profile.jpg** nahi daaloge, site par placeholder (circle icon) dikhega. Daalne ke baad sab jagah tumhari photo show hogi.

Preferred: 400x400 px ya usse bada, JPG ya PNG.
