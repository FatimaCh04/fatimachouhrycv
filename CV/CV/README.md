# CV / Portfolio Site (AWT)

Yeh folder portfolio website ka code hai.

- **Public pages:** index.html, about.html, services.html, portfolio.html, blog.html, resume.html, contact.html
- **Admin:** admin/ (login, profile, projects, services, blog – localStorage)
- **Assets:** assets/ (CSS, JS)

## Vercel par deploy

Repo ke root se nahi, is **CV** folder se deploy karna ho to Vercel Project Settings → Root Directory → `CV` set karo.
