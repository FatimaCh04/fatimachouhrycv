# Vercel se project delete karke dobara deploy

## Part 1: Project delete karna

1. **Vercel par login karo**  
   https://vercel.com

2. **Dashboard se apna project kholo**  
   Jo project delete karna hai (e.g. FA23-BSE-123-6B-Fatima-Ch-AWT) us par click karo.

3. **Settings open karo**  
   Upar **Settings** tab par click karo.

4. **Project delete karo**  
   - Neeche scroll karo  
   - **"Delete Project"** ya **"Remove Project"** section dhoondo  
   - **Delete** par click karo  
   - Confirm karo (project name type karke ya button dabao)

---

## Part 2: Dobara deploy karna

1. **Vercel Dashboard par jao**  
   https://vercel.com/dashboard

2. **Add New Project**  
   - **"Add New..."** button par click karo  
   - **"Project"** select karo

3. **GitHub repo select karo**  
   - List mein **FA23-BSE-123-6B-Fatima-Ch-AWT** dhoondo  
   - **Import** par click karo

4. **Root Directory set karo**  
   - **Root Directory** ke saamne **Edit** par click karo  
   - **CV** likho (sirf yeh)  
   - Save / Continue

5. **Deploy**  
   - **Deploy** button par click karo  
   - Jab build khatam ho, **Visit** se live site kholo

---

## Short summary

| Step | Kya karna hai |
|------|----------------|
| Delete | Project → Settings → scroll down → Delete Project → confirm |
| Dobara | Add New → Project → repo Import → Root Directory = **CV** → Deploy |

Iske baad naya live URL milega (purana wala kaam nahi karega).
