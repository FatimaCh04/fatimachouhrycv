# Vercel par site live karne ke step-by-step

## Step 1: Vercel par jao
- Browser mein open karo: **https://vercel.com**
- **Sign Up** ya **Log In** karo (Google / GitHub se login kar sakte ho)

---

## Step 2: Naya Project add karo
- Vercel ke dashboard par **"Add New..."** button dikhega (right side)
- Us par click karo
- List mein se **"Project"** select karo

---

## Step 3: GitHub repo select karo
- Ab screen par GitHub ki repositories list aayegi
- List mein **"FA23-BSE-123-6B-Fatima-Ch-AWT"** dhoondo
- Us repo ke saamne **"Import"** button hoga — us par click karo

*(Agar repo nahi dikhe to "Adjust GitHub App Permissions" ya "Configure GitHub App" par jakar is repo ko allow karo)*

---

## Step 4: Root Directory = CV set karo
- Import ke baad **"Configure Project"** wali screen aayegi
- Neeche **"Root Directory"** ka option hoga
- **"Root Directory"** ke saamne abhi **"./"** ya khali hoga
- Us box par click karo aur type karo: **CV**
- Ya **"Edit"** par click karke **CV** likho

Matlab: Vercel ko bolo ke site ka code **CV** folder se lo, poore repo se nahi.

---

## Step 5: Deploy karo
- **"Deploy"** button (blue/green) par click karo
- Thodi der wait karo — build ho rahi hogi
- Jab **"Congratulations"** ya **"Your project is live"** aaye, tab **Visit** par click karo

---

## Step 6: Live link
- Jo link milega (jaise `https://fa23-bse-123-6b-fatima-ch-awt.vercel.app`) wahi tumhari live site hai
- Is link ko kisi ko bhejo — portfolio open ho jayega

---

## Short summary
| Step | Kya karna hai |
|------|----------------|
| 1 | vercel.com → Login |
| 2 | Add New → Project |
| 3 | Repo **FA23-BSE-123-6B-Fatima-Ch-AWT** → Import |
| 4 | **Root Directory** mein **CV** likho |
| 5 | Deploy click karo |
| 6 | Visit se live site kholo |

---

## Agar Root Directory nahi dikhe
- **Configure Project** / **Build and Output Settings** section kholo
- Wahan **Root Directory** ka field hoga — usme **CV** daal do
